This repository corresponds to question 4 of the data reproducibility homework.

It contains raw and cleaned data from the Palmer Penguins dataset.

It contains an R script file used to produce a linear model of flipper length and culmen length in gentoos, and an associated graph. The graph is also saved as a pdf in 'figures'.

It contains one of the custom functions used in the R script, 'data.clean' which was used to clean the raw data.

The R script includes hashed comments aiming to walk the reader through the process of creating the contents of this repository.
